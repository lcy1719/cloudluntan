### 校园论坛nacos+SpringCloud+vue+万字文档
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、SpringCloud、nacos、Mybatis/MybatisPlus、GateWay、Vue、ElementUI、JavaScript、CSS、node.js、echarts、axisos

系统功能模块：
主要微服务分为三个（网关、认证和主要微服务）：GetwayApplication、CollegeApplication、SpringbootApplication
普通用户功能：
1，注册、登录、退出校园论坛网站
2，个人信息修改：修改登录密码、修改个人头像、电话、邮箱等功能
4，论坛推荐：推荐论坛帖子
5，校园论坛：搜索校园论坛、发布文章、查看论坛榜单、查看论坛详情
6，我的收藏：查看我收藏的帖子、取消收藏、搜索
7，查看帖子详情：点赞帖子、评论帖子、回复帖子、分享帖子到微博、收藏帖子
管理员功能：
1，主页：通过可视化页面统计用户数量、论坛文章数量、各季度用户统计
3，帖子帖子管理：查看系统所有帖子，搜索、编辑、删除、新增和查看帖子内容
4，帖子评论管理：查看系统所有评论，搜索、编辑和删除评论
5，帖子收藏管理：查看系统所有收藏，搜索、编辑和删除收藏
6，帖子分类管理：查看系统帖子的分类，搜索、编辑、新增和删除帖子分类
7，系统管理：包括用户管理、角色管理、菜单管理、文件管理
8，用户管理：多条件搜索用户，新增、编辑、删除用户，导入和导出用户
9，角色管理：新增用户角色、分配角色菜单、编辑角色、删除角色、搜索角色
10，帖子分类管理：新增帖子分类、编辑帖子分类、删除帖子分类、搜索帖子分类✔ 掌握Spring Cloud微服务架构搭建 ✔ Nacos服务注册与发现实践 ✔ Gateway网关控制 ✔ Vue与后端交互开发 ✔ 基础CRUD功能实现逻辑
